# Node.js快速安装指南（macOS）

## 推荐方法：直接从官方网站下载安装

### 步骤1：下载Node.js安装包

1. 打开浏览器访问官方中文网站：
   [https://nodejs.cn/download/](https://nodejs.cn/download/)

2. 在页面中找到**macOS 安装包**，点击下载最新的LTS版本（长期支持版）
   - 当前最新LTS版本：24.12.0
   - 下载文件名为：`node-v24.12.0.pkg`

### 步骤2：安装Node.js

1. 找到下载的安装包（通常在"下载"文件夹中）
2. 双击安装包文件
3. 按照安装向导的提示完成安装：
   - 点击"继续" → "继续" → "同意" → "安装"
   - 可能需要输入您的macOS用户密码
   - 安装过程通常只需1-2分钟

### 步骤3：验证安装

1. 安装完成后，重新打开一个新的终端窗口
2. 运行以下命令验证Node.js和npm是否安装成功：

```bash
node -v
npm -v
```

如果安装成功，您将看到类似以下的版本信息：
```
v24.12.0
10.8.2
```

### 步骤4：安装前端依赖

安装完成后，回到项目目录安装前端依赖：

```bash
cd /Applications/工作文件/Industrial-Detection-System-Vue-OpenCV-master
npm install
```

### 步骤5：启动前端服务

依赖安装完成后，启动前端开发服务器：

```bash
npm run dev
```

## 常见问题解决

### 问题：命令仍然无法找到

如果安装后运行`node -v`或`npm -v`仍然显示"command not found"：

1. **确保打开新的终端窗口**：安装后需要重新打开终端才能加载新的环境变量
2. **检查安装路径**：运行`which node`查看Node.js是否正确安装
3. **重启电脑**：如果上述方法都不行，尝试重启电脑后再试

### 问题：安装包无法打开

如果双击安装包显示"无法打开"：

1. 按住Control键点击安装包，然后选择"打开"
2. 在弹出的对话框中点击"打开"

## 安装完成

Node.js安装完成后，您的开发环境就准备好了！

- 后端服务：`python -m uvicorn src.backend.main:app --reload`
- 前端服务：`npm run dev`

现在您可以同时运行这两个服务来使用工业检测系统了！
